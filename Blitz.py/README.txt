1) Установите python, запустив файл python-3.4.4.amd64.msi  
		(желательно выбрать в одном из окон установщика пункт "Add to system path")
2) Установите PyQT , запустив файл PyQt4-4.11.4-gpl-Py3.4-Qt4.8.7-x64.exe
3) запустите файл start.bat

Если не заработало:
4) Правой кнопкой мыши кликните на файл blitz.py, выберите пункт Edit with IDLE
5) В октрывшемся окне, не трогая программный код, нажмите F5
